# Lab 7

This README provides instructions on how to execute the `lab_7.py` file.

## Prerequisites

Ensure you have the following installed:

- Python 3.x

## Instructions

1. **Navigate to the directory:**

    ```sh
    cd /path/to/lab_7
    ```

2. **Run the script:**

    ```sh
    python3 lab_7.py
    ```

## Troubleshooting

If you encounter any issues, ensure that:

- Python is correctly installed and added to your PATH.
- You are in the correct directory.
